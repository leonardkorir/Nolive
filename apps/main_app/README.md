# main_app

主应用壳工程，现已补齐 `android` / `ios` / `linux` / `macos` / `windows` 平台目录。

说明：

- `lib/` 仍由当前仓库维护，平台目录由 Flutter 模板托管。
- 如需重新生成平台模板，可在仓库根目录执行 `scripts/bootstrap_main_app_platforms.sh`。
- 当前默认显示名为 `Nolive`，bundle id / application id 为 `app.nolive.mobile`。
- 当前首发发布目标只锁定 Android mobile；非 Android 平台目录暂时只作为后续扩展保留。
